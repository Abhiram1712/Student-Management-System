# 🎓 Student Management System

A modern and user-friendly **Student Management System** designed to simplify student record management. This web-based application enables users to add, view, edit, and manage student information through an intuitive interface.

## 🌐 Live Demo

🔗 https://student-management-system-blue-chi.vercel.app

---


## 📖 Overview

The Student Management System is a frontend web application that provides essential student management functionalities through a clean and responsive user interface. The project demonstrates CRUD-style workflow design and basic authentication page structures commonly used in educational management platforms.

### 👤 User Authentication
- User Registration

### 📚 Student Management

## 🚀 Getting Started

### Prerequisites

You only need:

- A modern web browser
  - Google Chrome
  - Microsoft Edge
  - Mozilla Firefox
  - Safari

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Abhiram1712/Student-Management-System.git
```

2. Navigate to the project directory:

```bash
cd Student-Management-System
```

3. Open the project:

```bash
index.html
```

Or simply open `index.html` in your browser.
